# Pelatihan KOMDIGI VSGA 2025 - Junior Web Developer

Proyek ini merupakan bagian dari sertifikasi **Junior Web Developer** dalam program **Vocational School Graduate Academy (VSGA)** tahun 2025 oleh **KOMDIGI**.

## 🏢 Profil Perusahaan: PT. Bhumi Rantau Sinergi
Proyek ini bertujuan untuk membangun website profil sederhana bagi **PT. Bhumi Rantau Sinergi**, sebuah perusahaan yang bergerak di bidang pertambangan batu bara .

## 🛠️ Tech Stack & Tools
Proyek ini dibangun menggunakan kombinasi teknologi berikut:
* **Web:** PHP
* **Database:** SQL (MySQL via XAMPP/Laragon)
* **Editor:** Visual Studio Code
* **Server Lokal:** Laragon / XAMPP

## 🚀 Fitur Utama
* Halaman Landing Page (Profil Perusahaan)
* Informasi Layanan & Produk
* Manajemen Konten Sederhana
